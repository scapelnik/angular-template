Fakulteta za gradbeništvo in geodezijo (FGG)

Fakulteta za gradbeništvo in geodezijo Univerze v Ljubljani (UL FGG) je ena izmed
vodilnih izobraževalnih in raziskovalnih ustanov na področju gradbeništva,
geodezije in okoljskega inženirstva v Sloveniji.
FGG povezuje tradicijo, kakovostno izobraževanje in sodobne tehnologije ter
študentom omogoča pridobivanje znanja, ki je neposredno uporabno v praksi.

V tej GIS aplikaciji prikazujemo sodobno zgrajeno aplikacijo, ki temelji na WEB servisih.
Aplikacija omogoča vnos podatkov, urejanje podatkov in brisanje prostorskih podatkov.
V back-endu so Django servisi, ki črpajo podatke iz Postgres baze. Postgres baza ima GIS ekstenzijo za grafične podatke
Front-end je realiziran z Angularjem, ki prikazuje podatke na Open Layers karti.
Aplikacija je uporabniško prijazna, odzivna in preprosta za uporabo.

WEB DEVELOP AND GEOPORTALS

This is a simple web application for managing land registry data.
It allows users to insert, update, and delete records related to parcels, addresses, and roads.
The application is built using Angular and integrates with a PostgreSQL database.
The application uses a Open Layers map with OpenStreetMap tiles for spatial input.
The back-end is implemented using Django.
It is designed to be user-friendly and responsive for easy navigation.
