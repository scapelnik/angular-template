GeoSG

GeoSG is a demonstrational geoportal intended for displaying spatial data of the cadastral municipality of Slovenj Gradec. The project represents a practical example of using the latest and most commonly used web GIS technologies and shows how modern approaches can be applied in the development of clear, responsive and user-friendly WebGIS solutions.

The application combines a front-end cartographic display, server-side logic and a spatial database, which enables a comprehensive insight into the functioning of modern web-based spatial systems.

Technical information

GeoSG is built on an open-source technology stack that represents the standard in the development of modern GIS applications:

  • Angular – a modular and fast front-end framework for building responsive web applications
  • OpenLayers – a library for advanced cartographic visualisations and working with spatial layers
  • Django REST Framework – server infrastructure for data exchange via APIs
  • PostgreSQL/PostGIS – the leading open-source spatial database with advanced geometric functions

The application enables the input, editing, deletion and visualisation of spatial data (parcels, addresses, buildings and roads) and uses PostGIS for spatial queries, topological checks and cartographic rendering.

Purpose of the project

The aim of GeoSG is to show how modern open-source technologies can form an efficient, transparent and easily scalable web spatial solution. The project demonstrates:

  • linking a spatial database with a cartographic interface,
  • performing real-time data editing via REST APIs,
  • the use of advanced PostGIS functions for processing geometries,
  • the design of clear and practically oriented cartographic visualisations.
